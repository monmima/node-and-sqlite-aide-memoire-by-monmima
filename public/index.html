.ejs files

The page that you are currently reading is an HTML file. If you want to replace this index.html file by an index.ejs file, you would need to:

  1. Delete the index.html file
  2. Create a file with the name index.ejs in the views folder
  3. Make sure the route "/" is taken into account in the app.js file
